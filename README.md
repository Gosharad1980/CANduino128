# CANduino128
	- Break-out board pour la carte Olimex AVR-H128-CAN
	- Carte sur base d'AVR AT90CAN128
		- PC104 form factor
		- CAN network
		- SPI
		- I2C
		- bus d'addresse/Data externe via 74AHC373
